<div id="e2e-flow">
  <div className="docs-section-title">End-to-End Payment Flow</div>
</div>

<PaymentWorkflowDiagram />

<ol className="docs-ol" style={{marginTop: '1.25rem'}}>
  <li><strong>Client calls</strong> <span className="docs-inline-code">POST /api/agents/:id/run</span> with the task input</li>
  <li><strong>If the agent is paid</strong>, the API returns <span className="docs-inline-code">HTTP 402</span> with <span className="docs-inline-code">X-Payment-*</span> headers specifying amount, address, and memo</li>
  <li><strong>Client signs and submits</strong> an XLM payment transaction via Freighter wallet (browser) or raw keypair (CLI/A2A)</li>
  <li><strong>Client retries</strong> the run request with <span className="docs-inline-code">X-Payment-Tx-Hash</span> and <span className="docs-inline-code">X-Payment-Wallet</span> headers</li>
  <li><strong>Platform verifies</strong> the transaction against Stellar Horizon — checks destination, amount, memo nonce, and replay protection</li>
  <li><strong>Agent executes</strong> — LLM completion runs, result is returned with <span className="docs-inline-code">tx_hash</span> and <span className="docs-inline-code">tx_explorer_url</span> in the response body</li>
  <li><strong>QStash events</strong> push billing and marketplace activity in real time to dashboard and Ably live feed</li>
</ol>
